Installation

🐧 Linux :

  • Installez GCC en utilisant sudo apt-get install gcc
  • Installez un environnement de développement intégré (IDE), vous pouvez utiliser n'importe quelle application.
    • Je recommande Visual Studio Code pour les débutants, utilisez cette extension pour ajouter des aides et coloriser le code.

🍎 MacOS :

Les conseils de Raysan5 vous n'êtes pas obligés d'installer Raylib

🪟 Windows :

Les conseils de Raysan5 vous n'êtes pas obligés d'installer Raylib
